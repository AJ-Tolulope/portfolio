How it all started.

I figured out that there is a lot of ways to create your very own website, even without having any knowlegde of programming. One of the most popular way is the use of wordpress (website builder). With wordpress or any other website builder, creating a website has become easy, you can create your very own website with thousands of ready-made themes (designs) to choose from, changing how your website looks with a lot features.

Although website builders have there advantages. but they also have a lot of features to choose from, making it hard to get. So to master any website builder, it will take a while. Why go through all that when you can create any website of your choice with the help of HTML (hyper text markup language) CSS (cascading style sheet) and Javascript.

The amazing part is, unlike wordpress, you can create website with HTML, CSS and Javascript locally (offline) with your computer because they are all text files and can be edited with any text editor and then upload it live on the internet so people can have access to your website with your domain (website name) through a host (where your website will be uploaded to).

So i decided to learn every thing there is to learn about HTML, CSS and javascript.
